# 📝 To-Do List App Demo

## 📌 Project Title - To-Do List Application

## 📖 Project Overview
This project demonstrates a simple and efficient To-Do List application.  
It allows users to add, edit, delete, and mark tasks as complete, helping them stay organized and productive.

---

## 🏗️ Architecture
- **HTML** → Provides the structure of the app.  
- **CSS** → Styles the interface for a clean and responsive design.  
- **JavaScript** → Handles task logic, DOM manipulation, and local storage.  

**Flow:**  
User → UI (HTML/CSS) → JavaScript Logic → Local Storage → Rendered Task List

---

## 📂 Project Structure

project-root/ │── index.html        # Main UI │── style.css         # Styling and layout │── app.js            # Core functionality and logic │── README.md         # Documentation

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Local Storage API  

---

## 🔄 Workflow
1. User adds a new task.  
2. Task is stored in local storage.  
3. User can mark task as complete or delete it.  
4. Tasks persist even after closing the browser.  

---

## ▶️ How to Run (Step-by-Step)

  1. Clone the Repository

git clone https://github.com/pavitramudadla/To_Do_List_App.git


  2. Navigate to Project Folder
     
  cd To_Do_List_App


  3. Open in Browser

  Simply open index.html in your browser to start using the app.

🧪 Testing
- Add Task → Enter a task and click "Add".
- Complete Task → Click the checkbox to mark as done.
- Delete Task → Click the delete icon to remove.
- Persistence → Refresh the page; tasks remain saved.
📊 Output
- Task List View → Displays all tasks with status.
- Completed Task → Strikethrough style applied.
- Deleted Task → Removed from list and local storage.
