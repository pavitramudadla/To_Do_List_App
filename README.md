🚀 How to Run the Program

Follow the steps below to set up and run this project on your local machine:

📥 1. Clone the Repository
git clone https://github.com/pavitramudadla/To_Do_List_App.git
📂 2. Navigate to the Project Directory
cd To_Do_List_App
⚙️ 3. Install Dependencies

Make sure you have the required tools installed (like Python / Node.js depending on your project).

For Python projects:
pip install -r requirements.txt
For Node.js projects:
npm install
▶️ 4. Run the Application
For Python:
python app.py
For Node.js:
npm start

(Replace app.py or command based on your actual file name)

🌐 5. Open in Browser (if applicable)

If your project runs on a local server, open:

http://localhost:3000

or

http://127.0.0.1:5000

The application should now be running successfully.
